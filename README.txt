================================================================
     GOLOS 2.0 — Голосовой ввод текста для Windows
================================================================


  УСТАНОВКА
  ---------
  1. Распакуйте архив в любую папку (например C:\Golos2)
     ВАЖНО: путь к папке не должен содержать русских букв!
     Правильно:  C:\Golos2
     Неправильно: C:\Программы\Голос

  2. Запустите Golos2.exe

  3. Готово! В области уведомлений (трей, возле часов)
     появится серая иконка микрофона.


  КАК ПОЛЬЗОВАТЬСЯ
  ----------------
  * Нажмите Alt+X — начнется запись (иконка станет зеленой)
  * Говорите — текст вставится в любое активное окно
  * Нажмите Alt+X ещё раз — запись остановится

  Работает в любом приложении: браузер, мессенджер,
  текстовый редактор, ChatGPT, Claude и т.д.


  НАСТРОЙКИ (правый клик по иконке в трее)
  -----------------------------------------
  * Горячая клавиша — сменить комбинацию клавиш
  * Автозапуск — включить/выключить запуск при старте Windows
  * Выход — закрыть программу


  ГОЛОСОВЫЕ КОМАНДЫ (говорите во время записи)
  --------------------------------------------
  точка           .         запятая         ,
  вопрос          ?         восклицание     !
  двоеточие       :         тире            —
  новая строка    Enter     абзац           Enter+Enter
  открыть скобку  (         закрыть скобку  )
  кавычки         "         удали           удалить слово
  удали все       удалить всю строку

  Полный список: 40 команд (пунктуация, скобки, кавычки,
  спецсимволы, форматирование, удаление).


  СОВЕТЫ
  ------
  * Говорите четко, с небольшими паузами между фразами
  * Микрофон на расстоянии 20-40 см
  * Фоновый шум снижает точность
  * НЕ запускайте от имени администратора — иначе текст
    не будет вставляться в обычные окна


  ФАЙЛЫ В ПАПКЕ
  -------------
  Golos2.exe      — программа (запускать этот файл!)
  _internal/      — системные файлы программы (не удалять!)
  model/          — модель распознавания речи (не удалять!)
  dictionary.txt  — словарь автозамены (можно редактировать)
  config.json     — настройки (создается автоматически)
