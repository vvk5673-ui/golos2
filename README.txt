          GOLOS 2.0 — Голосовой ввод текста для Windows
              Набирай текст голосом. Быстрее в 3 раза.
═══════════════════════════════════════════════════════

Golos 2.0 работает в любом окне — браузер, мессенджер,
редактор, ChatGPT, Claude.
Нажал горячую клавишу, сказал текст, нажал ещё раз — готово.


───────────────────────────────────────────────────────
  ВОЗМОЖНОСТИ
───────────────────────────────────────────────────────

  * Голосовой ввод в любом приложении
  * Горячая клавиша Alt+X — включить/выключить
  * 40 голосовых команд — пунктуация, скобки, спецсимволы
  * Работает без интернета — данные не уходят в облако
  * Русский язык из коробки
  * Автозаглавная буква, умные пробелы вокруг знаков
  * Лёгкий и быстрый — работает в фоне


───────────────────────────────────────────────────────
  БЫСТРЫЙ СТАРТ
───────────────────────────────────────────────────────

  Шаг 1. УСТАНОВКА (один раз)

    Запустите файл install.bat
    Он автоматически установит всё необходимое.
    Требуется Python (скачать: https://python.org)

  Шаг 2. ЗАПУСК

    Запустите start.bat или ярлык "Golos 2.0" на рабочем столе.

  Шаг 3. ИСПОЛЬЗОВАНИЕ

    1. Откройте любое приложение (браузер, блокнот, чат)
    2. Поставьте курсор в текстовое поле
    3. Нажмите Alt+X — высокий сигнал (запись началась)
    4. Говорите — текст появляется в поле ввода
    5. Нажмите Alt+X ещё раз — низкий сигнал (стоп)

  ИКОНКА В ТРЕЕ
───────────────────────────────────────────────────────

  После запуска рядом с часами появляется иконка:

  * Серый круг            — ожидание (нажмите Alt+X)
  * Зелёный + красная точка — идёт запись

  Правый клик по иконке -> Выход — закрыть приложение.

───────────────────────────────────────────────────────
  ЧАСТЫЕ ВОПРОСЫ
───────────────────────────────────────────────────────

  Текст не набирается
    Убедитесь, что курсор стоит в текстовом поле.
    Попробуйте в Блокноте для теста.

  Плохо распознаёт
    Говорите чётко, не слишком быстро.
    Держите микрофон на расстоянии 20-40 см.

  Конфликт горячих клавиш
    Если Alt+X занята — закройте другую программу.

  Антивирус блокирует
    Добавьте папку golos2 в исключения антивируса.


───────────────────────────────────────────────────────
  СОВЕТЫ ДЛЯ ЛУЧШЕГО РАСПОЗНАВАНИЯ
───────────────────────────────────────────────────────

  1. Говорите чётко, произносите слова полностью
  2. Между предложениями делайте небольшую паузу
  3. Микрофон на расстоянии 20-40 см
  4. Фоновый шум снижает точность
  5. Голосовые команды произносите отдельно
