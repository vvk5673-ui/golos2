# Golos 2.0 — Голосовой ввод текста для Windows

**Набирай текст голосом в любом приложении. Быстрее клавиатуры в 3 раза.**

Golos 2.0 работает в любом окне — браузер, мессенджер, редактор, ChatGPT, Claude.
Нажал горячую клавишу, сказал текст, нажал ещё раз — готово.

---

## Возможности

- **Голосовой ввод в любом приложении** — работает везде, где можно печатать
- **Горячая клавиша Alt+X** — включить/выключить одним нажатием
- **40 голосовых команд** — пунктуация, скобки, спецсимволы, удаление
- **Работает без интернета** — все данные остаются на вашем компьютере
- **Русский язык** — модель распознавания заточена под русскую речь
- **Умная обработка** — автоматическая заглавная буква, правильные пробелы вокруг знаков
- **Лёгкий и быстрый** — работает в фоне, не мешает работе

## Быстрый старт

### 1. Установка (один раз)

Запустите файл **`install.bat`** от имени администратора.
Он автоматически установит все необходимые компоненты.

### 2. Запуск

Запустите файл **`start.bat`** или ярлык **Golos 2.0** на рабочем столе.

### 3. Использование

1. Откройте любое приложение (браузер, блокнот, чат)
2. Поставьте курсор в текстовое поле
3. Нажмите **Alt+X** — услышите высокий звуковой сигнал (запись началась)
4. Говорите — текст появляется в поле ввода
5. Нажмите **Alt+X** ещё раз — низкий сигнал (запись остановлена)

## Пользовательский словарь

Файл **`dictionary.txt`** — ваш личный словарь автозамены. Откройте его Блокнотом и добавьте слова, которые приложение распознаёт неправильно.

Формат: `неправильное = правильное` (одна замена на строку)

```
# мои замены
кот = код
промыт = промпт
нейро сеть = нейросеть
```

Строки с `#` — комментарии, они игнорируются. Словарь загружается при запуске приложения.

## Иконка в трее

После запуска в системном трее (рядом с часами) появляется иконка:

- **Серый круг** — приложение готово, ожидает нажатия Alt+X
- **Зелёный круг с красной точкой** — идёт запись

Правый клик по иконке → **Выход** — закрыть приложение.

## Частые вопросы

**Текст не набирается**
Убедитесь, что курсор стоит в текстовом поле. Попробуйте в Блокноте для теста.

**Плохо распознаёт**
Говорите чётко, не слишком быстро. Держите микрофон на расстоянии 20-40 см.

**Конфликт горячих клавиш**
Если Alt+X занята другой программой — закройте её или свяжитесь с автором.

**Антивирус блокирует**
Приложение использует библиотеку `keyboard` для горячих клавиш. Некоторые антивирусы могут на это реагировать. Добавьте папку golos2 в исключения.
