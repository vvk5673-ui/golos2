# Golos 2 — Полный список голосовых команд

## Знаки препинания

| Скажите | Результат | Пример |
|---|---|---|
| **точка** | . | «Привет точка» → Привет. |
| **запятая** | , | «Привет запятая как дела» → Привет, как дела |
| **вопрос** | ? | «Как дела вопрос» → Как дела? |
| **вопросительный** | ? | то же самое |
| **восклицание** | ! | «Отлично восклицание» → Отлично! |
| **восклицательный** | ! | то же самое |
| **двоеточие** | : | «Список двоеточие» → Список: |
| **точка с запятой** | ; | «Первое точка с запятой второе» → Первое; второе |
| **тире** | — | «Москва тире столица» → Москва — столица |
| **дефис** | - | «кто дефис то» → кто-то |
| **многоточие** | ... | «Ну многоточие» → Ну... |
| **троеточие** | ... | то же самое |

## Автоматическая обработка

Эти функции работают без голосовых команд:

| Функция | Что делает |
|---|---|
| Заглавная буква | Первая буква фразы автоматически становится заглавной |
| Пробелы перед знаками | Убираются: «слово .» → «слово.» |
| Пробелы после знаков | Добавляются: «слово.слово» → «слово. слово» |

## Советы для лучшего распознавания

1. **Говорите чётко** — не торопитесь, произносите слова полностью
2. **Делайте паузы** — между предложениями делайте небольшую паузу
3. **Микрофон** — держите на расстоянии 20-40 см
4. **Тишина** — фоновый шум снижает точность распознавания
5. **Команды** — произносите голосовые команды отдельно, с паузой
