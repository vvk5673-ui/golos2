# Анализ конкурентов — Голосовой ввод текста

> Дата анализа: 31 марта 2026

## 1. Wispr Flow
- **Сайт:** wisprflow.ai
- **Платформы:** Mac, Windows, iPhone, Android
- **Цена:** Бесплатно (2000 слов/нед) → $12/мес (год) / $15/мес (помесячно)
- **Технология:** Облако (требует интернет)
- **Точность:** ~97%
- **Языки:** 100+
- **Ключевые фишки:** Удаляет слова-паразиты ("эм", "ну"), AI-редактирование текста голосом, кроссплатформенная синхронизация, автопунктуация
- **Минусы:** Только облако, тяжёлый по ресурсам, нет офлайн-режима
- **Целевая аудитория:** Профессионалы, продуктивность

## 2. Voicy
- **Сайт:** usevoicy.com
- **Платформы:** Mac, Windows, браузер
- **Цена:** $8.49/мес / $82/год / $220 пожизненно
- **Технология:** Облако
- **Точность:** 99%+
- **Языки:** 50+
- **Ключевые фишки:** Работает в любом приложении через горячую клавишу, AI-редактирование выделенного текста голосом ("сделай профессиональнее"), автопунктуация
- **Минусы:** Требует интернет
- **Целевая аудитория:** Все, кто работает с текстом

## 3. Dragon Professional (Nuance)
- **Сайт:** nuance.com/dragon
- **Платформы:** Windows, Mac, мобильные
- **Цена:** $699 (единоразово) или $15-55/мес (облачная версия)
- **Технология:** Локально + облако
- **Точность:** 99%
- **Языки:** Основные европейские
- **Ключевые фишки:** Эталон индустрии, обучается под голос, медицинские/юридические словари, голосовые макросы
- **Минусы:** Очень дорогой, сложный, устаревший интерфейс
- **Целевая аудитория:** Корпорации, врачи, юристы

## Ключевые выводы

### 1. Где пустое место на рынке?
- **Офлайн + Windows + доступная цена** — SuperWhisper работает офлайн, но только на Mac. Dragon работает офлайн на Windows, но стоит $699. Никто не делает доступный офлайн-ввод для Windows.
- **Заточенность под промпты для нейросетей** — никто из конкурентов не позиционируется как инструмент для работы с AI. Все говорят "для писателей", "для продуктивности", но не для промптов.
- **Русский язык как приоритет** — большинство сервисов ориентированы на английский. Качество русского — вторично.

### 2. Наши преимущества (уже есть в golos!)
- ✅ Работает офлайн (Vosk) — данные не уходят в облако
- ✅ Windows — основная платформа
- ✅ Горячая клавиша (Alt+X) — работает в любом окне
- ✅ Голосовые команды (пунктуация, удаление)
- ✅ Русский язык из коробки
- ✅ Бесплатный (пока) — нет зависимости от облачных API

### 3. Чего не хватает для конкуренции
- ❌ AI-обработка текста (как у Wispr/Voicy — убрать паразиты, переформулировать)
- ❌ Автопунктуация (сейчас только голосовые команды)
- ❌ Высокая точность (Vosk ~90%, конкуренты 97-99%)
- ❌ Красивый интерфейс (сейчас — иконка в трее + консоль)
- ❌ Кроссплатформенность (только Windows)
- ❌ Пользовательский словарь

### 4. Рекомендуемая ценовая ниша
**$5-8/мес** — ниже Wispr Flow и Voicy, но с уникальным позиционированием:
- «Голосовой ввод для работы с нейросетями»
- Офлайн (приватность)
- Заточен под русский язык

### 5. Главный конкурентный вывод
> Рынок голосового ввода 2026 — это битва облачных сервисов за $10-15/мес.
> Ниша **офлайн + Windows + русский + промпты для AI + доступная цена** — практически пустая.
