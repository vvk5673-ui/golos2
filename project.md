# Golos 2.0 — Паспорт проекта

## Формула идеи
**Я создаю** приложение голосового набора текста
**Для** людей, которые ежедневно работают с текстом на компьютере
**Чтобы решить** проблему медленного набора, усталости рук и неудобства ввода длинных текстов

---

## Проблемы, которые решаем
1. **Медленный набор** — средняя скорость печати 150-200 символов/мин, голосом — 400-600
2. **Усталость рук** — при длительной работе за компьютером (туннельный синдром, боли в запястьях)
3. **Хочется диктовать на ходу** — не привязан к клавиатуре
4. **Заполнение промптов для нейросетей** — длинные, подробные запросы проще надиктовать, чем напечатать

## Выбор основной ЦА

### 🎯 Основная: Пользователи нейросетей
**Почему именно они:**
- Уже привыкли платить за цифровые инструменты ($20-200/мес)
- Промпты — идеальный use case для голосового ввода (длинные, подробные тексты)
- Рынок растёт экспоненциально
- Легко найти и достучаться (AI-сообщества, курсы, YouTube)
- Понимают ценность инструментов для продуктивности
- **Боль острая и ежедневная** — каждый день пишут десятки промптов

### 🎯 Дополнительная: Предприниматели + фрилансеры
Они пересекаются с основной ЦА (многие уже используют нейросети).

## Уникальное позиционирование (гипотеза)
**«Голосовой ввод, заточенный под работу с нейросетями»**

Не просто «диктовка текста», а инструмент для тех, кто каждый день общается с AI.

Возможные преимущества (уточнить после анализа конкурентов):
- Работает в любом окне (ChatGPT, Claude, любое приложение)
- Горячая клавиша — включил/выключил, текст сразу в поле ввода
- Голосовые команды (пунктуация, удаление, форматирование)
- Русский язык из коробки, без интернета (офлайн через Vosk)
- Простой и понятный — никаких сложных настроек
- Цена ниже конкурентов (уточнить после исследования)
